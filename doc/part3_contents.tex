\subsection{Multiplication Table}
\begin{frame}[fragile]
  \frametitle{Exercises \subpart{Multiplication Table}}
  \vspace{3mm}
  Steps:
  \begin{enumerate}
    \item From Python, produce an XLSX file that has one sheet.
    \item This sheet contains the 10s table.
    \item The cells containing the two factors should be in bold.
    \item the result cells should be defined through a formula over the factor cells.
    \item The diagonal should have a blue background. These are the square numbers.
  \end{enumerate}
\end{frame}

\subsection{Download Data}
\begin{frame}[fragile]
  \frametitle{Exercises \subpart{Download Data}}
  \vspace{3mm}
  Go to the UN data portal (\url{https://data.un.org}).
  
  \vspace{5mm}
  Under \textsl{\say{Popular statistical tables, country (area) and regional profiles}}, find the \textsl{\say{Population, surface area and density}} dataset.
  
  \vspace{5mm}
  Copy the link to the CSV version of this dataset (via right-click menu).
  
  \vspace{5mm}
  Make a program that downloads this link and stores the file locally as \texttt{popdata.csv}.
\end{frame}

\subsection{Process Data}
\begin{frame}[fragile]
  \frametitle{Exercises \subpart{Process Data}}
  \vspace{3mm}
  Make a new program that:
  \begin{enumerate}
    \item Reads \texttt{popdata.csv} from the last exercise.
    \item Process the data:
      \begin{enumerate}
        \item The second column contains an area descriptor. Drop all rows that do not have a country name in this row.
        \item The fourth column is the metric. Drop all rows that do not cover the \say{Population density} metric.
        \item The third column is the year. Drop all but the newest year that has the \say{Population density} metric.
        \item The fifth column has the value (i.e., the population density). We care about this.
      \end{enumerate}
    \item Based on the processed data, produce a data structure that maps country name to population density.
    \item Marshals the data structure to JSON format and store it as \texttt{popdata.json}.
  \end{enumerate}
\end{frame}

\subsection{Putting It All Together}
\begin{frame}[fragile]
  \frametitle{Exercises \subpart{Putting It All Together}}
  \vspace{0mm}
  The exercise is to produce an XLSX file with a sheet per continent. In each sheet should have a list of countries that belong to this continent (and their population densities). They should be listed in order of population density. The background color should act as a heatmap. That is the country with the highest population density (of the continent) should have a red background, and the one with the lowest should have a white background.
\end{frame}
